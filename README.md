# Nvim-configuration

## General Mappings

| Action       | Mapping     |
|--------------|-------------|
| Leader key   | `space`     |
| Escape key   | `jk`        |
| Close buffer | `Space + c` |
| Quit         | `Space + q` |
| Force quit   | `Ctrl + q`  |
| Save file    | `Space + w` |
